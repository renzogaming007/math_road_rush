# Math Road Rush

Math Road Rush is a kid-friendly arcade driving game where players steer into the lane with the correct math answer.

## How to Play

- Use `Left/Right Arrow` or `A/D` to move
- On mobile, use the on-screen left and right buttons
- Drive through the lane with the correct answer
- Correct answer: score +1
- Wrong answer: lose 1 life
- You start with 3 lives

## Game Modes

- `Easy` - simpler single-digit math
- `Intermediate` - mixed single- and double-digit math
- `Hard` - includes harder double-digit math

## Features

- 3-lane arcade gameplay
- Addition, subtraction, multiplication, and division
- Responsive for desktop and mobile
- Best score saved with localStorage
- Playable directly in the browser with one HTML file

## Play Locally

Just open `math_road_rush.html` in your browser.
